Språk


- HTML står för HyperText Markup Language och är en av webbens grundläggande byggstenar tillsammans med CSS (Cascading Style Sheets) och Javascript. - HTML utgör strukturen för alla internets webbsidor i form av rubriker, styckeindelning, att särskilja visst innehåll från annat och liknande. Det går också att infoga bilder med HTML. HTML är ett märkspråk och inte ett programmeringsspråk, eftersom HTML-kod endast har i uppgift att ge struktur. För att ett språk ska få kallas programmeringsspråk behöver det finnas möjlighet att utföra kommandon, vilket inte går med HTML. Varför är det egentligen viktigt att strukturera webbsidor, kan man fråga sig. En stor anledning är för att göra design och funktionalitet möjlig, vilket åstadkoms i ett samarbete mellan HTML och CSS respektive HTML och Javascript. Om HTML utgör strukturen kan man säga att CSS utgör designen och Javascript funktionaliteten på en webbsida.

- CSS står för Cascading Style Sheet. Det betyder kort och gott att det är en stilmall, också kallat stilmallsspråk. Likt HTML räknas detta inte som ett programmeringsspråk, då det saknar möjligheten att utföra riktiga kommandon. Samtidigt är CSS beroende av ett märkspråk som HTML för att göra något.
- CSS samlar alla instruktioner för hemsidans utseende i ett separat dokument, vilket gör att HTML-dokumentet bara innehåller hemsidans innehåll, något som är mycket positivt både en person som skapar en hemsida samt om en utomstående vill gå igenom hemsidans kod, eftersom det skapar struktur. Det är också väldigt bra ut sökmotoroptimerings-synpunkt, eftersom en sökmotor inte behöver gå igenom en massa ”utseende-kod” som den ändå inte bryr sig om.


Bootstrap är ett ramverk för att utveckla lyhörda och mobila vänliga webbplatser. Det är gratis och enkelt att ladda ner. Bootstrap är ett front-end-ramverk som hjälper enklare och snabbare webbutveckling. Den innehåller olika HTML- och CSS-baserade mallar för knappar, tabeller, blanketter, paneler och många fler.
Bootstrap är populär på grund av ett antal skäl. Det låter användarna skapa svåra webbplatser. En lyhörd webbplats kan automatiskt anpassa sig för att se bättre ut enligt enheten. Därför, om användaren visar webbplatsen från en smartphone eller ett skrivbord, visas webbplatsen densamma.
En Bootstrap innehåller en uppsättning funktioner. Den har ett avancerat nät system. Det innehåller ett antal återanvändbara komponenter för att skapa dropdowns, varningar, popup-fönster, navigationer etc. Användaren kan också innehålla många jQuery-plugins i projektet. Komponenterna och jQuery-pluginsna är anpassningsbara. Därför kan användaren ändra dem enligt hans val.

Java är ett programmeringsspråk och en datorplattform som först lanserades av Sun Microsystems år 1995. Det finns många applikationer och webbplatser som inte fungerar om du inte har installerat Java, och utbudet ökar för varje dag. Java är snabb, tillförlitlig och finns överallt, på datacentraler, i spelkonsoler, i superdatorer för forskningsändamål, i mobiltelefoner och på Internet! Java är ett programmeringsspråk och plattform för att skapa och köra datorprogram, så många olika enheter och operativsystem som möjligt skriva Java program Om du är en utvecklare som vill skapa program med hjälp av programmeringsspråket Java, så behöver du en kopia av både Java Runtime Environment och Java Development Kit på din dator för att fungera . Detta gäller även om du vill utveckla Java -program avsedda att köras i en webbläsare kallas applets. programmet kräver det. Den enklaste anledningen till att du skulle behöva Java på din dator beror på ett program du har ner och vill använda kräver Java Runtime Environment för att fungera . Dessa program skrevs av utvecklare som väljer Java som språk på grund av dess förmåga att skapa kod som kan köras på en så bred variation av enheter som möjligt , från mobiltelefoner till datorer , utan ändringar . Om programmet kräver Java att vara närvarande , kommer det att informera dig med ett felmeddelande när du försöker köra den .


Javascript är en av webbens tre grundläggande byggstenar tillsammans med HTML och CSS. Om HTML utgör strukturen och CSS designen, kan man säga att Javascript utgör funktionaliteten på internets webbsidor. Numera kan enkla animationer åstadkommas med hjälp av CSS, men generellt är det Javascript som möjliggör allt som "händer" på en webbsida. Exempel på saker som händer kan vara att någonting ändras när man klickar på en knapp, att ett formulär visar att jag har fyllt i mitt personnummer i fel format, eller att någonting rör på sig eller ändras på hemsidan. En vanlig missuppfattning är att Java och Javascript är samma sak, vilket inte stämmer. Båda är programmeringsspråk, men de är relativt väsensskilda vad gäller hur de används.
- Ett vanligt användningsområde för Javascript är att bestämma vad som ska hända när en användare interagerar på olika sätt på en webbsida. Det kallas för att använda "eventlyssnare" (eng. event listeners). Exempelvis går det att lyssna efter att en användare skrollar, klickar på ENTER-tangenten eller börjar fylla i ett formulär, och bestämma vad som ska hända när lyssnaren hör dessa saker.
